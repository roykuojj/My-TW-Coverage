# 3529 - 力旺

## 業務簡介
**板塊:** Technology
**產業:** Semiconductors
**市值:** 148,254 百萬台幣
**企業價值:** 144,793 百萬台幣

力旺電子 (3529) 是全球嵌入式[[非揮發性記憶體]] (NVM) [[矽智財]] (IP) 領導廠商，提供 OTP、MTP 及 [[eFlash]] 等 IP 授權予全球超過 2,400 家晶圓廠、IDM 及 IC 設計公司。核心產品包括 NeoBit (OTP)、NeoFuse (Anti-fuse)、NeoFlash (eFlash) 及 [[RRAM]]/[[MRAM]] IP。已與[[台積電]]合作至 2 奈米製程 IP 驗證，自 2010 年起每年獲頒台積電最佳 IP 夥伴獎。

## 供應鏈位置
**上游 (IP 開發):**
- **EDA 工具:** [[Cadence]]、[[Synopsys]]

**中游 (IP 授權):**
- **NVM IP:** **力旺** (全球 NVM IP 龍頭，2,400+ 授權客戶)
- **資安子公司:** PUFsecurity (安全 IP)

**下游 (晶圓廠與 IC 設計):**
- **晶圓代工:** [[台積電]] (2nm IP 驗證)、[[聯華電子]] (22nm RRAM)
- **授權客戶:** [[三菱電機]]、[[Toshiba]]、[[世界先進]]

## 主要客戶及供應商
### 主要客戶 (IP 授權)
- **晶圓代工:** [[台積電]] (最佳 IP 夥伴獎)、[[聯華電子]]、[[世界先進]]
- **IDM:** [[三菱電機]]、[[Toshiba]]

### 主要供應商
- **EDA:** [[Cadence]]、[[Synopsys]]
- **競爭同業:** [[Synopsys]] (嵌入式記憶體 IP)
