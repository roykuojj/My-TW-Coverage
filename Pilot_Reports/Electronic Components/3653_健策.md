# 3653 - 健策

## 業務簡介
**板塊:** Technology
**產業:** Electronic Components
**市值:** 464,309 百萬台幣
**企業價值:** 452,431 百萬台幣

健策精密 (Kenmec) 為台灣最大的 **均熱片 (Vapor Chamber)** 供應商，也是全球精密沖壓及[[散熱]]解決方案的領導者。
公司產品廣泛應用於[[伺服器]]、[[AI]] 運算、車用電子及通訊領域。
核心優勢與亮點：
1.  **[[AI 伺服器]]供應鏈**: 為 [[AMD]]、[[Intel]] 及 [[NVIDIA]] 的關鍵供應商，供應其 GPU 與 CPU 所需的高階均熱片與[[散熱]]模組。
2.  **散熱技術領先**: 具備冷鍛均熱片、真空腔[[散熱]]技術，並積極佈局 [[AI]] [[伺服器]]所需的 **液冷** [[散熱]]模組與微通道蓋板 (MCL)。
3.  **伺服器扣件 (ILM)**: 具備[[伺服器]]處理器[[連接器]]緊固件 (ILM) 的設計與製造能力，在[[伺服器]]市場具備極高市佔。
4.  **自動化能量**: 具備自有[[自動化]]設備開發團隊，支援生產線高度[[自動化]]，提升產品良率與交期競爭力。

## 供應鏈位置
*   **上游**: 銅材、導熱材料、電子組件原料。
*   **中游**: **健策 (3653)** - 均熱片、ILM 扣件、[[散熱]]模組設計製造。
*   **下游**:
    *   **處理器大廠**: [[AMD]], [[Intel]], [[NVIDIA]].
    *   **伺服器代工廠**: [[鴻海]], [[廣達]], [[緯創]].
    *   **車用與通訊**: 汽車 Tier-1 供應商, 基站設備業者。

## 主要客戶及供應商
### 主要客戶
*   知名[[半導體]]巨頭：[[AMD]], [[Intel]], [[NVIDIA]]。
*   [[雲端服務]]供應商 ([[CSP]]) 的 [[AI]] 加速器專案。
*   歐系與美系車用電子客戶。
