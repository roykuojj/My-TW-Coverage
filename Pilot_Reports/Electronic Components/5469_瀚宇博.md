# 5469 - 瀚宇博

## 業務簡介
**板塊:** Technology
**產業:** Electronic Components
**市值:** 45,773 百萬台幣
**企業價值:** 76,195 百萬台幣

瀚宇博德 (HannStar Board) 為 **華新麗華集團 (PSA)** 旗下 [[PCB]] 大廠，全球最大 **Notebook PCB** 供應商。
產品組合持續優化：
1.  **PC/NB**: 全球市佔第一，但佔比隨市場成熟而調整。
2.  **伺服器/網通**: 高層數板、[[伺服器]]背板、[[AI]] Server 主板 (OAM/UBB)。
3.  **遊戲機**: 為主要遊戲機大廠供應商。
4.  **車用**: 併購 **精成科 (6191)** 後強化車用 [[PCB]] 佈局。

## 供應鏈位置
*   **中游 (PCB製造)**: **瀚宇博 (5469)** - 硬板 (Rigid [[PCB]]), HDI.
*   **上游**:
    *   **基板**: [[銅箔]]基板 (CCL) - 聯茂, 台光電.
    *   **材料**: [[玻纖布]], [[銅箔]].
*   **下游**:
    *   **代工廠 (ODM)**: [[廣達]], 仁寶, [[緯創]], [[鴻海]].
    *   **品牌**: [[Dell]], [[HP]], [[Microsoft]] (Xbox), [[Sony]] (PS).

## 主要客戶及供應商
### 主要客戶
*   **NB/PC**: [[廣達]], 仁寶, [[緯創]].
*   **遊戲機**: [[Sony]], [[Microsoft]].
*   **伺服器**: 品牌[[伺服器]]廠與雲端客戶.

### 主要供應商
*   **CCL**: 南亞, 聯茂.
