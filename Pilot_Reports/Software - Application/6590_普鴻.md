# 6590_普鴻

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 1,785 百萬台幣
**企業價值:** 1,597 百萬台幣

普鴻資訊 (6590，Provision) 為台灣金融支付與資訊安全軟體系統的隱形冠軍。公司專注於金融業核心支付應用、清算系統及企業級資安解決方案，在台灣銀行圈的票交/跨行支付系統市佔率極高。有別於一般軟體外包商，普鴻主打自有產品授權與常態性維護服務 (SaaS/PaaS)，創造穩定的高黏著度經常性收入。核心產品 ProTrust 資料保護平台涵蓋透明資料庫加密、資料去識別化、企業金鑰管理及檔案加密等功能，搭配自主研發並通過 FIPS 140-2 Level 3 國際認證的 [[HSM]] (硬體加密模組)。公司透過併購捷智商訊進一步稱霸金融監理科技 (RegTech) 法規報表領域。

## 供應鏈位置
**上游 (技術平台與元件):**
- **資料庫/軟體:** [[Microsoft]]、[[Oracle]] (資料庫引擎)
- **雲端平台:** [[AWS]]、[[Microsoft]] Azure (公有雲 IaaS)
- **資安硬體:** [[Thales]] (國際 HSM 硬體加密模組合作)

**中游 (金融資安軟體):**
- **支付/資安系統:** **普鴻資訊** (金融電子支付清算平台、ProTrust 資安防護、RegTech 法規報表)

**下游 (終端客戶):**
- **商業銀行:** [[富邦金]]、[[玉山金]]、[[台新金]]等幾乎所有大型銀行
- **純網銀:** [[LINE Bank]]、[[樂天銀行]]
- **政府金融機構:** 財政部、中央銀行
- **第三方支付:** 國內第三方支付業者
