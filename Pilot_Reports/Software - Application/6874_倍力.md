# 6874_倍力

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 1,451 百萬台幣
**企業價值:** 1,652 百萬台幣

倍力資訊 (6874，M-Power) 專注於企業級軟體代理與自主研發 ESG 數位轉型解決方案。公司代理 [[Oracle]]、[[SAP]]、[[Cisco]]、[[IBM]] 等國際頂級商用軟體品牌，掌握極佳之企業客戶覆蓋率。近年成功轉型，自主研發「集團合併報表系統 (GCRS)」及「財務附註揭露系統 (FNDS)」協助上市櫃公司快速產出 [[IFRS]] 合併報表，以及「碳管家 (CarbonKeeper)」企業碳盤查與減碳模擬系統。倍力精準切入台灣數千家上市櫃公司必須強制接軌國際永續準則與財報合規的剛性需求，累計服務超過 400 家上市櫃企業。
