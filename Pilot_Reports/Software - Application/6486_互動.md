# 6486_互動

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 4,005 百萬台幣
**企業價值:** 3,331 百萬台幣

互動國際數位 (6486，IDT) 隸屬於[[佳世達]]集團，為台灣頂尖的電信、媒體及寬頻網路系統整合商。公司專注於代理歐美一線硬體與軟體品牌，為客戶提供 [[5G]] 骨幹網路、微波無線傳輸、數位電視多媒體播送、地理資訊系統 (GIS) 及雲端資安之完整系統建置與售後服務。主要代理品牌包括 [[Cisco]]、[[Ericsson]]、[[ESRI]] (全球 GIS 軟體龍頭) 及 [[AWS]] 等國際大廠。隨著 [[5G]] 企業專網、[[低軌衛星]]通訊及智慧城市蓬勃發展，互動國際已將業務延伸至高鐵軌道通訊、智慧城市防災等大型基礎設施智慧化建置。

## 供應鏈位置
**上游 (國際品牌原廠):**
- **網通設備:** [[Cisco]]、[[Ericsson]] (5G/微波通訊設備)
- **GIS 軟體:** [[ESRI]] (全球 GIS 軟體龍頭)
- **雲端平台:** [[AWS]] (雲端基礎設施)
- **多媒體設備:** 國際數位電視製播軟硬體原廠

**中游 (系統整合):**
- **電信/媒體 SI:** **互動國際** (電信網路、數位媒體、GIS 系統整合建置)

**下游 (終端客戶):**
- **電信營運商:** [[中華電信]]、[[台灣大哥大]]、[[遠傳電信]]
- **媒體:** 各大有線/無線電視台、OTT 新媒體平台
- **交通:** [[台灣高鐵]] (軌道通訊系統)
- **政府:** 各縣市政府 (智慧城市/防災系統)
