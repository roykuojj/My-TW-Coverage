# 3494 - 誠研

## 業務簡介
**板塊:** Technology
**產業:** Computer Hardware
**市值:** 844 百萬台幣
**企業價值:** 1,822 百萬台幣

誠研科技 (3494，[[HiTi Digital]]) 為數位影像輸出解決方案供應商，以自有品牌 HiTi 行銷全球。核心技術為熱昇華 (Dye-Sublimation) 列印技術，主要產品包括相片印表機 (Photo Printer)、證卡列印機 (Card Printer) 及相關耗材 (色帶/相紙)。

公司經營「立可得」(Likoda) 自助沖印機台品牌，佈局台灣 [[7-ELEVEN]]、[[全家便利商店]] 等超商與賣場通路。近年積極開拓證卡列印市場（身分證/員工證/會員卡），以及商業活動即時拍照列印服務，擴大耗材訂閱收入之經常性營收佔比。

## 供應鏈位置
**上游 (列印核心零組件):**
- **熱寫頭:** 日系熱感應列印頭供應商
- **馬達/IC:** 精密馬達與控制 IC
- **色帶原料:** 熱昇華色帶原物料
- **相紙:** 專用相片紙與證卡卡材

**中游 (列印設備與耗材製造):**
- **相片列印:** **誠研** (HiTi 品牌熱昇華相片印表機)
- **證卡列印:** **誠研** (HiTi 品牌 ID 卡/會員卡印表機)
- **自助沖印:** **誠研** (立可得 Likoda 自助沖印機台)

**下游 (列印應用市場):**
- **超商通路:** [[7-ELEVEN]]、[[全家便利商店]] — 立可得自助沖印
- **證件製發:** 政府機關與企業 — 證卡列印機
- **商業攝影:** 婚紗/活動攝影業者 — 相片印表機
- **全球經銷:** 全球相片沖印設備經銷通路
