# 3434 - 哲固

## 業務簡介
**板塊:** Technology
**產業:** Computer Hardware
**市值:** 1,049 百萬台幣
**企業價值:** 1,139 百萬台幣

哲固資訊 (3434，[[ACULA]]) 專注於視訊顯示器 (Display) 產品設計與製造，主要產品線為安全監控顯示器 (CCTV Monitor)、工業電腦顯示器 (Industrial Display) 及醫療顯示器。

安全監控應用佔營收大宗，廣泛應用於安防系統與交通監控。近年積極拓展高毛利之醫療用顯示器（手術室/診斷用）及智慧病房解決方案。主要市場為北美與歐洲，以 ODM 模式服務國際安控品牌與醫療設備商。公司具備從光學校正、系統整合到量產之完整能力。

**上游 (面板與驅動元件):**
- **LCD 面板:** [[友達]]、[[群創]] — 工業/醫療級顯示面板
- **驅動 IC:** 顯示器驅動 IC 供應商
- **機構件:** 金屬/塑膠外殼射出件
- **電源模組:** 顯示器電源供應模組

**中游 (顯示器系統整合製造):**
- **安控顯示:** **哲固** (CCTV 安全監控顯示器)
- **醫療顯示:** **哲固** (手術室/診斷用醫療顯示器)
- **工業顯示:** **哲固** (工業電腦顯示器)

**下游 (安控與醫療終端):**
- **安控品牌:** 北美國際安控設備大廠 — CCTV 顯示器 ODM
- **醫療設備:** 歐美醫療器材品牌 — 手術室/診斷顯示器
- **工業電腦:** 工業電腦廠 — 搭配 IPC 之工業顯示器
