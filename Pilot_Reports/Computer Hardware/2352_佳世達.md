# 2352 - 佳世達

## 業務簡介
**板塊:** Technology
**產業:** Computer Hardware
**市值:** 32,854 百萬台幣
**企業價值:** 125,912 百萬台幣

佳世達科技 (2352，[[Qisda Corporation]]) 成立於 1984 年，總部位於桃園，為[[明基]] ([[BenQ]]) 集團核心公司，以設計製造服務 (DMS) 起家，現已轉型為橫跨資訊、醫療、智能方案、網路通訊及材料科學之大艦隊集團。

五大事業群：(1) DMS 代工：顯示器、投影機、網路攝影機；(2) 品牌行銷：[[BenQ]] 品牌顯示器、投影機、dvLED 顯示方案；(3) 醫療服務：經營兩家大型綜合醫院，醫療耗材及設備之 ODM/OEM，醫療事業近年成長 19%；(4) 智能方案：智慧企業、智慧製造、智慧教育、AI 邊緣運算 ([[Edge AI]])，成長達 20%；(5) 網通事業：LAN/WAN 網通設備及行動寬頻產品。旗下重要子公司包括[[明泰科技]] (Alpha Networks，網通)、[[友通資訊]] (DFI，嵌入式 IPC)、Partner Tech (POS 系統) 等。

## 供應鏈位置
**上游 (面板與電子元件):**
- **面板:** [[友達]] (AUO)、[[群創]] — 液晶面板/dvLED 模組
- **光學引擎:** 投影機光學引擎 (DLP/LCD 投影技術)
- **電子元件:** [[MLCC]]、IC 晶片、記憶體模組

**中游 (DMS 設計製造與集團事業):**
- **DMS 代工:** **佳世達** (顯示器/投影機/攝影機代工)
- **品牌事業:** [[BenQ]] (顯示器/投影機/dvLED 品牌)
- **醫療事業:** **佳世達** (兩家綜合醫院、醫療耗材設備)
- **智能方案:** **佳世達** (智慧製造/教育/[[Edge AI]])

**下游 (終端客戶與通路):**
- **消費通路:** 全球 IT 通路商 — BenQ 品牌產品
- **醫療機構:** 醫院及醫療器材經銷商
- **企業客戶:** 製造業及教育機構 — 智能方案
- **網通品牌:** 透過子公司[[明泰科技]]服務全球網通客戶
