# 2201 - 裕隆

## 業務簡介
**板塊:** Consumer Cyclical
**產業:** Auto Manufacturers
**市值:** N/A 百萬台幣
**企業價值:** N/A 百萬台幣

裕隆汽車 (2201，Yulon Motor) 為台灣歷史最悠久之汽車集團，與[[日產汽車]] (Nissan) 長期合資合作。核心業務涵蓋：(1) 整車組裝 — 苗栗三義廠為台灣主要汽車組裝基地；(2) 電動車合資 — 與[[鴻海集團]]共同創立[[鴻華先進]] (2258，鴻海持股 45.62%、裕隆 43.83%)，開發 Model C/B/E/T 等[[MIH]]平台電動車；(3) 品牌轉讓 — 已將自有品牌[[納智捷]] (LUXGEN) 100% 股權以 7.876 億元轉讓予[[鴻華先進]]。集團版圖涵蓋[[裕隆日產]] (2227)、[[中華汽車]] (2204)、[[裕融]] (9941，[[格上租車]])，另開發[[裕隆城]]新店大型商場。

## 供應鏈位置
**上游 (技術與平台):**
- **日系技術:** [[日產汽車]] — Nissan 原廠技術授權與套件
- **EV 平台:** [[鴻華先進]] (2258) — [[MIH]] 電動車平台與三電技術
- **動力系統:** [[日本電產]] (Nidec) — 三合一動力系統

**中游 (組裝製造):**
- **裕隆汽車** (苗栗三義廠整車組裝，為[[鴻華先進]] Model C 代工)

**下游 (品牌與服務):**
- **品牌銷售:** [[裕隆日產]] (2227) — Nissan/Infiniti 台灣行銷
- **商用車:** [[中華汽車]] (2204) — Mitsubishi 製造
- **汽車金融:** [[裕融]] (9941) / [[格上租車]] — 車輛融資與租賃
- **EV 品牌:** [[鴻華先進]] / [[納智捷]] — 電動車品牌
- **不動產:** [[裕隆城]] — 新店大型商場

## 主要客戶及供應商
### 主要客戶
- **EV 代工:** [[鴻華先進]] — Model C (n7) 整車代工
- **品牌銷售:** [[裕隆日產]] — Nissan 台灣行銷
- **一般消費者:** 台灣新車買家

### 主要供應商
- **技術授權:** [[日產汽車]] — Nissan 原廠技術
- **EV 平台:** [[鴻華先進]] — 電動車三電系統
- **動力系統:** [[日本電產]] (Nidec) — 三合一動力
